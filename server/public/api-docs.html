WebSocket Connection Details

Connection URL:

ws://localhost:3000/ws/interview?sessionId=YOUR_SESSION_ID&tokenId=YOUR_TOKEN_ID&mode=default

Query Parameters:

  • sessionId (required): Unique session identifier
  • tokenId (optional): Authentication token
  • mode (optional): Interview mode (default/coding)

Message Types:

  • Client → Server: audio_chunk, audio_end, text_answer, code_submission, cancel_turn
  • Server → Client: transcript_partial, transcript_final, ai_text_chunk, tts_audio_chunk, state_change
